# HealthcareManagmentApp

This repository contains the codebase for a Health Care Management Application. It provides functionality for users to manage their health-related documents and policies.

## Prerequisites

Before running the application, ensure you have the following installed:

- Ruby (version 2.7.5)
- Ruby on Rails (version 7.1.3.2)
- PostgreSQL

## Getting Started

To get started with HealthcareManagmentApp, follow these steps:

1. Clone this repository to your local machine:

   git clone https://github.com/Tanay-ror/healthcare_managment.git

2. Make sure you have mentioned ruby and rails version on your machine.

3. Install all dependecies by running:

    ` bundle install `

4. Create database using following command:
    ` rails db:create`

5. Run all Migrations using Following commands:
    ` rails db:migrate `


## Testing App

1. For testing app you can run following command on terminal:
      ` rspec `
